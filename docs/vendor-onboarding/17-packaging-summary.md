## The three invariants (everything obeys these)

1. **Per-repo graphs for evidence; unified graph only for breadth, frequency, and confidence.**

3. **We package graph-grounded context, templates, architecture, implementation guidance, and an agent handoff pack ‚Äî we do not auto-generate your production system.**
